### 1. MVP 게임 3종 (우선순위순)

#### 🎮 게임 1: 입 벌리기 게임 (구강운동)
| 항목 | 내용 |
|------|------|
| **치료 목표** | 턱 운동 범위(ROM) 확대, 입술 협응력 향상 |
| **핵심 동작** | 입 크게 벌리기 / 입술 오므리기 / 입꼬리 올리기 |
| **인식 기술** | MediaPipe Face Landmarker (입 벌림 각도, 입술 간 거리) |
| **난이도 3단계** | Level 1: 입 벌리기만 / Level 2: 벌리기+오므리기 교대 / Level 3: 3동작 연속 |
| **성공 기준** | 치료사가 아동별로 목표 각도 문턱값(threshold) 설정 |
| **세션 시간** | 3~5분 (집중력 한계 고려) |

#### 🎮 게임 2: 소리 열기구 (발성 훈련)
| 항목 | 내용 |
|------|------|
| **치료 목표** | 자발적 발성 유도, 발성 지속시간 확대, 음량 조절 |
| **핵심 동작** | 마이크에 대고 "아~" 소리 → 열기구가 올라감 |
| **인식 기술** | Web Audio API (음량 dB 실시간 감지, 지속시간 측정) |
| **난이도 5단계** | L1: 소리 내기(유무) / L2: 2초 유지 / L3: 4초 유지 / L4: 음량 조절 / L5: 음높이 매칭 |
| **성공 기준** | 발성 유무 → 지속시간(초) → 음량 범위(dB) 순서로 확장 |
| **세션 시간** | 5~10분 |

#### 🎮 게임 3: 따라 말하기 (조음 훈련)
| 항목 | 내용 |
|------|------|
| **치료 목표** | 음소 정확도 향상, 음절→단어→구 확장 |
| **핵심 동작** | 그림 카드 보고 따라 말하기 → 발음 정확도 점수 |
| **인식 기술** | Web Speech API (음성 인식) + 발음 스코어링 |
| **난이도** | 음소(ㄱ,ㄴ,ㄷ...) → 음절(가,나,다) → 단어(사과,바나나) → 2어절 구 |
| **성공 기준** | 정확도 60% → 70% → 80% 단계적 상향 (치료사 설정) |
| **세션 시간** | 5~10분 |

---

### 2. 절대 원칙 (🟠 정소리 강조)

> ⚠️ 이것만은 꼭 지켜주세요.

1. **실패에 감점 없음** — 틀려도 벌점이 아니라 "다시 해볼까?" 격려
2. **리더보드 없음** — 경쟁은 장애 아동에게 스트레스. 자기 자신과의 비교만
3. **치료사가 문턱값 설정** — 모든 성공 기준은 아이마다 다르게 설정 가능
4. **세션 데이터 기록 필수** — 발성 횟수, 지속시간, 정확도, 반응시간 모두 저장
5. **3~10분 세션** — 발달장애 아동 집중력 한계 존중
6. **부모 가이드** — 가정에서 따라할 수 있는 안내 제공 (일반화 전략)

### 3. 측정 지표 (IEP 추적용)

| 지표 | 게임 | 단위 |
|------|------|------|
| 자발적 발성 시도 횟수 | 소리 열기구 | 회/세션 |
| 발성 지속 시간 | 소리 열기구 | 초(sec) |
| 평균 음량 | 소리 열기구 | dB |
| 입 벌림 도달률 | 입 벌리기 | % (목표 대비) |
| 구강운동 성공 횟수 | 입 벌리기 | 회/세션 |
| 음소 정확도 | 따라 말하기 | % |
| 단어 수준 정확도 | 따라 말하기 | % |
| 세션 완료율 | 전체 | % |
| 연속 사용일 | 전체 | 일 |

1. **큰 터치 영역** — 모든 버튼 최소 56×56px (아동 손가락 크기)

3. **글자 최소화** — 아이콘 + 음성 안내 중심 (글 못 읽는 아동 고려)
4. **뒤로가기 항상 있음** — 실수해도 빠져나올 수 있게
5. **보상은 시각+청각 동시** — 파티클 효과 + 효과음 + 진동
