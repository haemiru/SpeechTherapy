# □ 사업화 과제 개요(요약)

## 1. 사업화 과제 소개

### 과제 개요

| 항목 | 내용 |
|------|------|
| **과제명** | AI 엣지컴퓨팅 기반 발달장애 아동 언어치료 게임 플랫폼 "소리야 놀자!" |
| **서비스 형태** | 웹 기반 SaaS (Progressive Web App) |

| **개발 단계** | MVP Phase 1 완료 (입 운동 게임 구현), Phase 2~4 개발 중 |

### 서비스 개요

"소리야 놀자!"는 발달장애 아동을 위한 **웹 기반 AI 언어치료 게임 플랫폼**입니다. 스마트폰 또는 태블릿의 카메라와 마이크만으로 **구강운동 → 발성 → 조음**의 임상적 치료 순서를 따르는 4종의 치료 게임을 제공합니다. 모든 AI 처리는 사용자 기기에서 수행되어(Edge AI) 개인정보가 외부로 전송되지 않으며, 매 세션마다 정량화된 치료 데이터를 생성하여 개별화교육프로그램(IEP) 목표 추적이 가능합니다.

[화면캡처 삽입: 홈 화면 — 4종 게임 카드가 배치된 메인 화면]

### 주요 특징 및 차별성

| 특징 | 설명 |

# 1. 문제인식(Problem)

## 1-1. 제품·서비스의 개발동기

### 외부동기: 발달장애 아동 언어치료의 구조적 위기

#### (1) 치료 접근성의 심각한 불균형

국내 등록 발달장애인은 약 273,000명이며, 이 중 8세 이하 아동은 13,806명(보건복지부, 2025.02)입니다. 그러나 이들이 적시에 적절한 언어치료를 받기는 매우 어렵습니다.

**치료 인력 부족**: 국내 언어재활사는 약 18,000명(2024년 기준)이나, 소아 전문 언어재활사는 이 중 약 30% 수준입니다. 특히 비수도권 지역의 경우 언어재활사 1인당 담당 아동 수가 수도권 대비 3~5배에 달해, 대기 기간만 3~6개월이 소요되는 실정입니다.

**비용 장벽**: 전문 언어치료는 회당 50,000~100,000원으로, 주 2~3회 치료 시 월 40~120만원의 비용이 발생합니다. 발달재활서비스 바우처(월 22만원)로는 주 1회 치료비도 충당하기 어렵습니다.

| 구분 | 수도권 | 비수도권 | 비고 |
|------|--------|---------|------|
| 언어재활사 밀도 | 1:150 | 1:450~750 | 아동 대비 |
| 평균 대기 기간 | 1~2개월 | 3~6개월 | 첫 치료까지 |
| 회당 치료비 | 50,000~80,000원 | 50,000~100,000원 | |
| 바우처 충당률 | 약 55% | 약 35% | 월 실제 치료비 대비 |

#### (2) 골든 타임의 구조적 상실

발달장애 아동의 언어치료 골든 타임은 **12~36개월**이지만, 국내 평균 진단 연령은 4세 이상입니다. 진단 후에도 치료 대기, 비용 문제, 지역 접근성 부족으로 실제 치료 시작까지 추가 6개월~1년이 소요됩니다.

```
[골든 타임 대 실제 치료 시작 시점 비교]

골든 타임:  ████████████████████ (12개월 ~ 36개월)
실제 진단:                              ▼ (평균 48개월+)
치료 시작:                                    ▼ (평균 54~60개월+)
                                              └── 골든 타임 이미 지남
```

#### (3) 가정 내 '일반화' 실패 문제

언어치료의 핵심 목표는 치료실에서 습득한 기술을 **가정과 일상**에서 자연스럽게 사용하는 '일반화'입니다. 그러나 현실은:

- 주 1~2회 치료로는 충분한 반복 연습 불가 (언어 습득에는 매일 반복 필요)
- 부모가 가정에서 치료 활동을 재현하기 어려움 (전문 지식 부족)
- 가정 연습의 객관적 측정·기록 수단 부재
- 치료사와 부모 간 세션 데이터 공유 시스템 미비

> **학술 근거**: "Digital games for rehabilitation of speech disorders in children" (PMC Scoping Review)에 따르면, 디지털 게임 기반 언어치료는 **치료 빈도 증가, 동기 유지, 가정 내 일반화 촉진**에서 유의미한 효과를 보였습니다.

#### (4) 디지털치료제(DTx) 시장의 급성장

국내 디지털치료제 시장은 2025년 약 1,200억원에서 2028년 약 3,500억원으로 **연평균 42% 성장**이 전망됩니다(KHIDI). 특히 아동 발달장애 분야는 정부의 "발달장애인 생애주기별 지원 강화" 정책과 맞물려 디지털 헬스케어 전환이 가속화되고 있습니다.

### 내부동기: 팀의 기술 역량과 임상 경험의 결합

#### (1) 엣지 AI 기술의 성숙

Google의 MediaPipe Face Landmarker는 **478개 얼굴 랜드마크 + 52개 블렌드셰이프**를 **30fps 이상**으로 실시간 추적할 수 있으며, 이 중 jawOpen(입 벌림), mouthPucker(입 오므리기), tongueOut/tongueUp/tongueLeft/tongueRight(혀 운동) 등의 블렌드셰이프가 구강운동 치료에 직접 활용 가능합니다.

이 기술의 핵심은 **100% 온디바이스 처리**라는 점입니다. 카메라 영상이 서버로 전송되지 않아 아동의 개인정보 보호가 완벽하게 보장되며, 인터넷 연결이 불안정한 환경에서도 사용할 수 있습니다.

```
[엣지 AI 처리 파이프라인]

카메라 입력 → MediaPipe Face Landmarker → 블렌드셰이프 추출 → 치료 목표 판정
  (기기 내)      (기기 내, GPU 가속)        (기기 내)         (기기 내)
                                                              │
                                                              ▼
                                                     세션 데이터 저장
                                                      (로컬 스토리지)
                                                              │
                                                              ▼
                                                     선택적 동기화
                                                      (사용자 동의 시)
```

#### (2) 언어재활사의 임상 주도 설계

본 서비스는 초기 기획 단계부터 **언어재활사(SLP)가 치료 목표, 난이도 체계, 성공 기준, 세션 구성**을 직접 설계했습니다. 이는 단순한 '교육용 게임'이 아닌 **임상 근거 기반 치료 도구**로서의 신뢰성을 확보하는 핵심 차별점입니다.

| 설계 영역 | SLP 주도 결정 사항 |
|-----------|-------------------|
| 게임 순서 | 구강운동(입+혀) → 발성 → 조음 (표준 치료 순서) |
| 난이도 체계 | 각 게임 3~5단계, 아동별 문턱값 개별 설정 |
| 성공 기준 | 절대 수치가 아닌 아동별 상대 목표 (IEP 연동) |
| 세션 시간 | 3~10분 (발달장애 아동 집중력 한계 존중) |
| 보상 정책 | 실패 감점 없음, 리더보드 없음, 긍정적 피드백만 |
| 데이터 추적 | 10개 이상 IEP 지표 자동 기록 |

#### (3) 웹 기술 기반 접근성 극대화

네이티브 앱이 아닌 **웹(PWA) 기반** 서비스를 선택한 이유:

- **설치 장벽 제거**: 앱 스토어 검색·다운로드·설치 과정 없이 URL 접속만으로 사용
- **기기 제한 없음**: iOS, Android, Windows, ChromeOS 등 브라우저가 있는 모든 기기 지원
- **업데이트 즉시 반영**: 앱 업데이트 대기 없이 서비스 배포 즉시 사용자에게 반영
- **비용 효율**: 앱 스토어 수수료(30%) 절감, 단일 코드베이스 유지

## 1-2. 제품·서비스의 목적(필요성)

### 솔루션 정의

"소리야 놀자!"는 **AI 엣지컴퓨팅 기술**을 활용하여 발달장애 아동이 **가정에서 매일, 부모와 함께** 언어치료 활동을 수행할 수 있도록 하는 **웹 기반 언어치료 게임 플랫폼**입니다.

### 주요 목적

| 목적 | 설명 | 해결 문제 |
|------|------|----------|
| **치료 빈도 증가** | 주 1~2회 → 매일 3~10분 가정 연습 | 반복 부족으로 인한 학습 정체 |
| **비용 장벽 해소** | 기본 무료, 프리미엄 월 9,900원 | 회당 5~10만원 치료비 부담 |
| **지역 격차 해소** | 인터넷만 있으면 어디서든 사용 | 비수도권 치료 접근성 부족 |
| **치료 데이터 정량화** | 매 세션 자동 데이터 기록 | 주관적 평가 의존, IEP 추적 어려움 |
| **가정 일반화 촉진** | 부모 가이드 + 게이미피케이션 | 치료실-가정 간 기술 전이 실패 |
| **조기 개입 확대** | 진단 즉시 사용 가능 | 치료 대기 시간으로 인한 골든 타임 상실 |

### 대상 아동

| 대상 | 연령 | 주요 증상 | 본 서비스 활용 영역 |
|------|------|----------|-------------------|
| 자폐스펙트럼장애(ASD) | 3~10세 | 언어 발달 지연, 발성 부족, 구강 감각 이상 | 발성 유도, 구강운동, 모방 훈련 |
| 지적장애 | 3~10세 | 언어 이해·표현 지연, 조음 오류 | 구강운동, 발성, 조음 전 단계 |
| 뇌성마비(CP) | 3~10세 | 구강운동 제한, 연하장애, 발성 제한 | 턱 ROM 확대, 입술 협응, 발성 |
| 뇌병변장애 | 3~10세 | 구강 운동 범위 제한, 혀 운동 약화 | 턱·혀·입술 운동 범위 확대 |
| 언어발달지연(DLD) | 3~7세 | 또래 대비 표현 언어 지연 | 발성 → 음절 → 단어 확장 |

### 개입 방식: 4종 치료 게임

본 서비스는 **표준 언어치료 순서**에 따라 4종의 게임을 순차적으로 제공합니다:

```
[임상 치료 순서]

Phase 1: 구강운동 기반 확립
┌─────────────────────┐     ┌─────────────────────┐
│ 🎮 게임 1: 입 운동   │ ──→ │ 🎮 게임 2: 혀 운동   │
│ 턱 ROM 확대         │     │ 혀 신장/거상/좌우     │
│ 입술 협응력 향상     │     │ 조음 기관 준비        │
│ ✅ 구현 완료         │     │ 🔜 개발 중           │
└─────────────────────┘     └─────────────────────┘
         │                           │
         ▼                           ▼
Phase 2: 발성 능력 발달
┌─────────────────────┐
│ 🎮 게임 3: 소리 열기구│
│ 자발적 발성 유도      │
│ 발성 지속시간 확대    │
│ 음량 조절 능력       │
│ 🔒 계획              │
└─────────────────────┘
         │
         ▼
Phase 3: 조음(발음) 정확도
┌─────────────────────┐
│ 🎮 게임 4: 따라 말하기│
│ 음소 → 음절 → 단어    │
│ 발음 정확도 향상      │
│ 한국어 음운체계 기반   │
│ 🔒 계획              │
└─────────────────────┘
```

#### 게임 1: 입 운동 (✅ 구현 완료)

| 항목 | 내용 |
|------|------|
| **치료 목표** | 턱 운동 범위(ROM) 확대, 입술 협응력 향상 |
| **핵심 동작** | 입 크게 벌리기 (jawOpen) / 입술 오므리기 (mouthPucker) |
| **인식 기술** | MediaPipe Face Landmarker — jawOpen, mouthPucker 블렌드셰이프 |
| **난이도** | Level 1: 벌리기만 / Level 2: 벌리기+오므리기 교대 / Level 3: 3동작 연속 |
| **세션 구성** | 5라운드, 라운드당 10초, 라운드 간 휴식 2초, 총 약 3~5분 |
| **성공 기준** | 아동별 문턱값 설정, 0.5초 이상 유지 시 성공 판정 |
| **측정 지표** | 턱 개방 도달률(%), 최고값, 유지시간(ms), 라운드 성공률 |

[화면캡처 삽입: 입 운동 게임 플레이 중 — 카메라 뷰에 게이지 오버레이가 표시된 화면]

#### 게임 2: 혀 운동 (🔜 개발 중)

| 항목 | 내용 |
|------|------|
| **치료 목표** | 혀 신장(protrusion), 거상(elevation), 좌우 이동(lateralization) |
| **핵심 동작** | 혀 내밀기 / 혀 올리기 / 혀 좌우 이동 |
| **인식 기술** | MediaPipe — tongueOut, tongueUp, tongueLeft, tongueRight 블렌드셰이프 |
| **난이도 5단계** | L1: 내밀기만 → L2: 내밀기+되돌리기 → L3: 올리기 → L4: 좌우 → L5: 복합 |
| **세션 구성** | 5라운드, 라운드당 10초, 라운드 간 휴식 3초 (혀 근육 피로 고려) |
| **임상 주의** | 혀 근육은 턱보다 피로가 빠름, 과도한 혀 내밀기 제한(최대 0.85), 구역 반사 주의 |
| **측정 지표** | 혀 신장 도달률, 거상 도달률, 좌우 대칭 비율, 최대 유지시간 |

#### 게임 3: 소리 열기구 (🔒 계획)

| 항목 | 내용 |
|------|------|
| **치료 목표** | 자발적 발성 유도, 발성 지속시간 확대, 음량 조절 |
| **핵심 동작** | 마이크에 "아~" 소리 → 열기구 상승 (음량 비례) |
| **인식 기술** | Web Audio API (음량 dB 실시간 감지, 지속시간 측정) |
| **난이도 5단계** | L1: 발성 유무 → L2: 2초 유지 → L3: 4초 유지 → L4: 음량 조절 → L5: 음높이 매칭 |
| **세션 구성** | 5라운드, 라운드당 8초, 5~10분 |
| **측정 지표** | 발성 시도 횟수, 발성 지속시간(초), 평균 음량(dB) |

#### 게임 4: 따라 말하기 (🔒 계획)

| 항목 | 내용 |
|------|------|
| **치료 목표** | 음소 정확도 향상, 음절→단어→구 확장 |
| **핵심 동작** | 그림 카드 보고 따라 말하기 → 발음 정확도 점수 |
| **인식 기술** | Web Speech API (음성 인식) + 자체 발음 스코어링 알고리즘 |
| **난이도** | 음소(ㄱ,ㄴ,ㄷ) → 음절(가,나,다) → 단어(사과,바나나) → 2어절 구 |
| **세션 구성** | 5라운드, 라운드당 15초, 5~10분 |
| **측정 지표** | 음소 정확도(%), 단어 정확도(%), 세션 완료율 |

### 기대 효과

| 영역 | 기대 효과 | 측정 방법 |
|------|----------|----------|
| **치료 빈도** | 주 1~2회 → 주 5~7회 (매일 연습) | 앱 사용 로그 |
| **구강운동 ROM** | 턱 개방 20% 이상 개선 (8주 후) | jawOpen 블렌드셰이프 최고값 추이 |
| **혀 운동 범위** | 혀 신장·거상 도달률 30% 이상 개선 | tongueOut/tongueUp 추이 |
| **발성 능력** | 발성 지속시간 평균 2초 → 5초 이상 | Web Audio API 지속시간 기록 |
| **조음 정확도** | 목표 음소 정확도 60% → 80% | Web Speech API 인식률 |

| **부모 참여** | 가정 훈련 실시율 80% 이상 | 부모 대시보드 활동 기록 |
| **비용 절감** | 가구당 월 치료비 30~50% 절감 | 설문조사 + 실제 치료비 비교 |

### 기술개발의 목적

본 과제의 기술개발 목적은 다음 3가지로 요약됩니다:

**첫째, 엣지 AI 기반 구강운동 정량 평가 기술 확립.**
MediaPipe Face Landmarker의 블렌드셰이프 데이터를 활용하여 턱·입술·혀의 운동 범위를 실시간으로 정량 측정하고, 이를 IEP 목표와 연동하는 자동 평가 시스템을 구축합니다.

**둘째, 임상 근거 기반 게이미피케이션 프레임워크 개발.**
언어재활사가 설계한 치료 목표·난이도·성공 기준을 게임 메커니즘(강아지 성장, 별 보상, 스트릭)과 결합하여, 아동의 치료 동기를 유지하면서도 임상적 유효성을 확보하는 프레임워크를 개발합니다.

**셋째, 치료사-부모-아동 연결 데이터 플랫폼 구축.**
가정 연습 데이터를 치료사에게 실시간 공유하고, 치료사가 원격으로 난이도를 조정하며, 부모에게 맞춤 가정 훈련 가이드를 제공하는 3자 연결 플랫폼을 구축합니다.
